# 🚀 Job Board - Full Stack Application

A modern job marketplace connecting freelancers with clients, built with .NET and Next.js.

## ✨ Features

- 🔐 **JWT Authentication** - Secure login/register
- 👥 **Role-based Access** - Client & Freelancer dashboards  
- 💼 **Job Management** - Post, browse, and apply to jobs
- 📊 **Application Tracking** - Accept/reject applications
- 🎨 **Modern UI** - NextUI + Tailwind CSS
- 🏗️ **Clean Architecture** - Repository pattern with AutoMapper

## 🛠️ Tech Stack

**Backend (.NET 8)**
- ASP.NET Core Web API
- Entity Framework Core
- JWT Authentication
- AutoMapper
- SQL Server

**Frontend (Next.js)**  
- TypeScript
- NextUI Components
- Tailwind CSS
- Shadcn Ui
